

# תוכנית עדכון תפריט

## סיכום
עדכון מלא של מאגר הנתונים מהאקסל החדש, הוספת עמודה חדשה "דבר השף" למאגר ולממשקים, ותיקון RTL בפאנל הניהול.

## שלב 1: הוספת עמודת "דבר השף" למאגר הנתונים
- הוספת עמודה `chef_note` (טקסט, nullable) לטבלת `dishes` באמצעות migration

## שלב 2: עדכון מלא של הנתונים
- מחיקת כל המנות והקטגוריות הקיימות
- הוספת כל הקטגוריות מהאקסל כולל קטגוריות חדשות:
  - ראשונות, סלטים, פסטה, מהטאבון, המיוחדים של השף, דגים, קינוחים, שתייה חמה, קוקטיילים, בירות, משקאות קלים, כוסות יין, יינות אדומים, יינות לבנים, רוזה, מבעבעים, מיוחדים
  - קטגוריות חדשות: ילדים - פיצות, ילדים - פסטה, ללא גלוטן (ראשונות/סלטים/מהטאבון/המיוחדים/דגים), טבעוני (ראשונות/מהטאבון/עיקריות)
- הוספת כל המנות עם מחירים מעודכנים ועמודת "דבר השף"
- עדכון קישורי תמונות מ-Google Drive לפורמט ישיר

## שלב 3: עדכון ממשק המשתמש (DishCard)
- הוספת תצוגת "דבר השף" בכרטיס המנה - מוצג בלחיצה על המנה (dialog/expandable)
- כשלוחצים על מנה, נפתח חלון עם פרטי המנה כולל "דבר השף"

## שלב 4: עדכון ממשק הניהול (Admin)
- תיקון RTL: כותרות ("קטגוריות"/"מנות") בצד ימין, כפתורים ("הוסף") בצד שמאל
- הוספת שדה "דבר השף" בטופס עריכת/הוספת מנה
- ודא ש-`flex-row-reverse` עובד נכון בכל המקומות

## פרטים טכניים

### Migration SQL
```sql
ALTER TABLE dishes ADD COLUMN chef_note text DEFAULT '';
```
